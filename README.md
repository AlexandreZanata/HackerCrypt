# HackerCrypt
HackerCrypt "A client-side encryption tool with hacker aesthetics featuring Morse, Base64, Caesar, Hex, Leet and AES-256 ciphers - all processing happens in your browser with no data transmission"

Here are the README files in both English and Portuguese for your GitHub repository:

# H4CK3R CRYPT0 - Encryption Tool

![Hacker Theme](https://img.shields.io/badge/Theme-Hacker-green) 
![No Backend](https://img.shields.io/badge/Backend-None-red)

A stylish hacker-themed encryption/decryption tool built with HTML, CSS (Tailwind), and React (via CDN). Works entirely client-side with no backend required.

## Features

- **Multiple Encryption Methods**:
  - Morse Code
  - Base64
  - Caesar Cipher (with adjustable shift)
  - Hexadecimal
  - Hacker Substitution (leet speak)
  - AES-256 (military-grade encryption)

- **Hacker Aesthetic**:
  - Glitch effects
  - Matrix background
  - Terminal-like interface
  - Neon colors

- **Client-Side Processing**:
  - No data leaves your browser
  - No server required

## Usage

1. Open `index.html` in any modern browser
2. Select encryption or decryption mode
3. Choose an algorithm
4. Enter your text
5. Click "Encrypt" or "Decrypt"

For AES-256, make sure to save your encryption key as it's required for decryption.

## Technical Details

- Built with React via CDN (no build step required)
- Uses Tailwind CSS for styling
- Includes custom animations and effects
- All processing happens in the browser
- AES implementation uses Web Crypto API
